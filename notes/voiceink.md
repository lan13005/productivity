## Building from source

We cannot update the software with this approach! If you do, you have to rebuild and likely update below permissions again.
```bash
git clone https://github.com/Beingpax/VoiceInk.git
cd VoiceInk
make local
open ~/Downloads/VoiceInk.app
```

## Permissions
You may have to go into VoiceInk settings > Permissions > Delete (-) old VoiceInk to Add(+) new one at location

## Audio Input
System Default - migrates between laptop and bluetooth devices
